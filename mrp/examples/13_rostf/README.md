# ROS TF2 Example

To run:
```
mrp up --reset_logs && mrp logs --old rostf
```

You should see something like:
```
rostf | b'header: \n'
rostf | b'  seq: 0\n'
rostf | b'  stamp: \n'
rostf | b'    secs: 0\n'
rostf | b'    nsecs:         0\n'
rostf | b'  frame_id: "a"\n'
rostf | b'child_frame_id: "c"\n'
rostf | b'transform: \n'
rostf | b'  translation: \n'
rostf | b'    x: 2.0\n'
rostf | b'    y: 0.0\n'
rostf | b'    z: 0.0\n'
rostf | b'  rotation: \n'
rostf | b'    x: 0.0\n'
rostf | b'    y: 0.0\n'
rostf | b'    z: 0.0\n'
rostf | b'    w: 1.0\n'
```
