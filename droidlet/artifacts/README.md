# Droidlet artifacts

The layout of this folder is:
```
- droidlet
  - artifacts
    - datasets
      - <right now this only has NLU datasets> But we will refactor this soon.
    - models 
      - nlu <one NLU model shared by all at the moment>
      - perception
        - craftassist <craftassist instance segmentation model>
        - locobot <locobot perception models>
```
    
## How to contribute
If you'd like to contribute to this folder, please follow instructions [here](../tools/artifact_scripts/README.md).
